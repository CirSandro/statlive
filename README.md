# StatLive

Projet scolaire réalisé dans le cadre de la matière **Veille Technologique**.

## Objectif du projet

L’objectif était de produire et partager, sur le support de notre choix, une **documentation technique** sur un sujet libre.

Notre sujet : **« La collecte de statistiques lors de matchs de football en temps réel »**.

Le rendu prend la forme d’un site web accessible ici : **https://statlive.vercel.app/**

## À propos de ce dépôt

Ce dépôt contient le code du site *StatLive*.

- Application front-end en **TypeScript**
- Déployée sur **Vercel**

## Lancer le projet en local

### Prérequis

- **Node.js** (version récente recommandée)
- **npm**

### Installation

```bash
npm install
```

### Démarrage en mode développement

```bash
npm run dev
```

### Build (production)

```bash
npm run build
```

### Aperçu du build

```bash
npm run preview
```
